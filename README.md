## Quick Start

1. Clone this repo
2. Run the tool for your use case:
    1. AKS:
        * Export your credentials
        * `make run-aks`
    1. EKS:
        * `make run-eks`
    1. GKE:
        * `make run-gke`
    1. K8s:
        * `make run-k8s`
